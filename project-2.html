🎮 Project Overview

The Rock Paper Scissors game is a simple and classic hand game played between two participants — the player and the computer. The game includes three choices: Rock, Paper, or Scissors, and follows standard rules:

Rock beats Scissors | Scissors beats Paper | Paper beats Rock

🎯 Objective

The objective of this project was to create an engaging and interactive web-based game where users can:

- Play against the computer with instant feedback

- Practice basic UI interaction using HTML, CSS, and JavaScript

⚙ How It Works

User Interaction: The user selects Rock, Paper, or Scissors by clicking a button.

Computer’s Move: JavaScript randomly selects the computer’s choice using Math.random().

Comparison: The two choices are compared using game logic.

Result Display: The winner (user or computer) is shown instantly on the screen.

💻 Technologies Used

HTML: For page structure and UI layout

CSS: For styling buttons, layout, and responsive design

JavaScript: To manage gameplay logic, random moves, and display results

🏁 Conclusion

This project helped me gain confidence in frontend technologies by building a small but fully interactive web app. It reinforced my understanding of DOM manipulation, event handling, and basic game logic.
